
# Prueba técnica Cysnet

Este es un pequeño proyecto típo e-commerce hecho con React donde puedes explorar productos, filtrarlos, visualizarlos y añadirlos a un carrito.

La idea ha sido practicar una arquitectura limpia y escalable usando herramientas modernas como React, Zustand para el estado global y React Query para la gestión de datos.

Incluye funcionalidades como búsqueda, filtrado por categoría y valoración, vista de detalle de producto con reseñas y gestión del carrito en tiempo real. La interfaz está pensada para ser simple, rápida y agradable de usar.

🔴 **IMPORTANTE:** El proyecto está completamente en inglés (nombres de carpetas, textos y estructura) para mantener coherencia con la API utilizada, que también está en inglés.

### Prueba local

Clonar el proyecto

```bash
  git clone https://github.com/probablynotannie/prueba_tecnica_react
```

Entrada proyecto

```bash
  cd prueba_tecnica_react
```

Instalación de dependencias

```bash
  npm install
```

Inicio de servidor

```bash
  npm run dev
```
